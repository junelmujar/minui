About MINUI

A minimalist UI framework loosely inspired by Material Design Lite and Bootstrap front-end frameworks. The project name came from the words "minimalistic" & "user interface", thus the name MINUI.


Why another UI library?

First and foremost, I built the library as an exercise / experiment. To see if I can build a front-end framework from scratch. The goal was to emulate the best bits of Material Design and Bootstrap, albeit in a more simplified form. As a developer who relied on jQuery for the Javascript bits on my work, I always wondered if I can do the same with plain Javascript. For sure it can be done 100%, as evidenced by other front-end frameworks out there. The challenge is if "I" or me can do it. And MINUI was born.


Current Status

The project is still in it's early stages and most of the components work. It still needs lots of documentation, samples and code refactoring.
