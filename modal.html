Modals

Modal Triggers

Creating elements that toggle modal(s) is easy via use of data- attributes.

Use the following attributes to configure your modal trigger:

  • data-toggle

    Identifies that the element is a modal trigger

  • data-target

    The modal to be toggled

  • data-close-esc

    Close the modal when Esc key is pressed

Events

The following custom events are emitted when toggling or dismissing a modal. They are dispatched to the document event manager.

  • minui.modal.open
  • minui.modal.dismissed
  • minui.modal.overlay.dismissed
  • minui.modal.esc.dismissed

			

	document.addEventListener('minui.modal.open', function(event) {
		console.log('Modal opened', event.detail.target);
	});
	document.addEventListener('minui.modal.dismissed', function(event) {
		console.log('Dismissed by close button', event.detail.target);
	});
	document.addEventListener('minui.modal.overlay.dismissed', function(event) {
		console.log('Dismissed by overlay', event.detail.target);
	});
	document.addEventListener('minui.modal.esc.dismissed', function(event) {
		console.log('Dismissed by Esc key', event.detail);
			});
