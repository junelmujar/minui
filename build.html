Build Tools

MINUI uses Gulp to compile the javascript and less files. It relies on the following packages:

Package List
  • babel-cli
  • babel-loader
  • gulp-babel-minify
  • gulp-clean-css
  • gulp-concat-css
  • gulp-less
  • gulp-rename
  • webpack-stream

Install

npm install gulp -G
npm install gulp-cli
npm install

To build the source, type the following command:

gulp build

To enable auto-build when hacking away on the code:

gulp watch
